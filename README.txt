### DESCRIPTION OF THE DIRECTORIES ###

=> Code
	This directory contains all the matlab code created.
	Further explanation are in this directory inside the README.txt file.


=> in - Lina
	This directory contains all the Lina marche converted from *.c3d
	to *.trc and *.mot.


=> in - Sirine
	This directory contains all the Sirine marche converted from *.c3d
	to *.trc and *.mot.


=> processedData - Lina
	This directory contains all the Lina marche Filled and Filtered.


=> processedData - Sirine
	This directory contains all the Sirine marche Filled and Filtered.


=> ScaledModelsOpenSim
	This directory contains the scaled model of Lina and Siline.
